# Geyavo Admin Frontend

Interface utilisateur moderne construite avec Vue 3, TypeScript et Tailwind CSS.

## Installation

```bash
pnpm install
```

## Développement

```bash
pnpm run dev
```

## Build Production

```bash
pnpm run build
pnpm run preview
```

## Structure

- `src/assets/` - CSS, images, fonts
- `src/components/` - Composants réutilisables
- `src/views/` - Pages de l'application
- `src/router/` - Configuration des routes
- `src/stores/` - Gestion d'état Pinia
- `src/services/` - Appels API
- `src/types/` - Types TypeScript
- `src/utils/` - Fonctions utilitaires

## Technologies

- Vue 3 (Composition API)
- TypeScript
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- Axios
- Lucide Vue (icônes)
